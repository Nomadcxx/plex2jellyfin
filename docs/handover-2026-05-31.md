## 4. Critical Bug Found & Fixed During Testing

### Infinite Recursion in `renameWithFallback()`
**Location:** `internal/housekeeping/engine.go:101`
**Root Cause:** During bucket 4 implementation, a sed replacement that swapped `os.Rename` calls to `e.renameWithFallback()` also replaced the call *inside* the helper itself.

**Before (broken):**
```go
func (e *Engine) renameWithFallback(src, dst string) error {
    if err := e.renameWithFallback(src, dst); err == nil {  // ← infinite recursion
        return nil
    }
```

**After (fixed):**
```go
func (e *Engine) renameWithFallback(src, dst string) error {
    if err := os.Rename(src, dst); err == nil {  // ← calls os.Rename first
        return nil
    } else if !errors.Is(err, syscall.EXDEV) {
        return err
    }
    // Cross-device fallback to transfer.Move...
```

**Validation:**
```bash

# PASS
```
