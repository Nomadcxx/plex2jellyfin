## 7. Key Architectural Decisions

1. **Map reset removal**: `loggedErrors` and `transientWarned` maps no longer reset at 2000 entries. A 2000-entry map is ~100KB — acceptable memory cost vs. log burst risk.

2. **Replay synchronization**: Deferred operation replay now runs synchronously instead of spawning goroutines. This serializes replay per path, preventing unbounded concurrency.

3. **Timer shutdown ordering**: `Shutdown()` now: cancels context → waits for `shutdownWg` → stops timers → waits for `timerWg` (in-flight callbacks) → closes resources. This prevents use-after-close.

4. **Transfer error propagation**: `Move()` now returns an error when source removal fails, so callers can distinguish "file copied but source still exists" from full success.

5. **Scheduler panic recovery**: `fire()` now recovers from panics, logs them, and records a failed run — preventing a single panicking job from crashing the daemon.

6. **Privilege escalation**: `Escalate()` uses `os.Executable()` instead of `os.Args[0]`, preventing PATH hijacking before sudo.
